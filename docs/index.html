Privacy-first stroke-recovery companion for families and caregivers.

Inspired to help families navigating stroke recovery, Web3Care (pre-MVP) is a PWA for tracking vitals, meds, meals, therapy, mood, appointments, bill payments, and cognitive-strengthening activities (games)—plus a daily non-PHI SMS summary. Solana-first (~$0.91/user/year). Releases mirrored to Storj (optional IPFS).

Features

HIPAA-minded design
Least-privilege, client-side encryption, auditability, vendor BAAs where applicable.
Care tracking that sticks
BP, meds, therapy, meals, sleep, mood, notes, appointments—now with bill payments and cognitive games built in.
Daily SMS summaries
Plain-language check-ins (no PHI in SMS). Tap to view securely in the app.
Seedless sign-in
Email 2FA + Passkeys (Face ID/Touch ID). No seed phrases.
Solana at scale
Low-fee receipts/roles via account abstraction & session keys.
Versioned releases
Frontend builds hosted on Storj for verifiable versions (optionally IPFS).
