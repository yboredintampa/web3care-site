Web3Care — Concept Deck

Inspired to help families navigating stroke recovery. Pre-MVP.

01 — Problem

Care is fragmented across vitals, meds, meals, therapy, and appointments. Families need simple routines and shared visibility without sacrificing privacy.

02 — Solution

Privacy-first PWA to track daily care, with a single-screen dashboard and a daily non-PHI SMS summary. Seedless sign-in (email 2FA + passkeys).

03 — Product

  • Big-button flows: BP, Meds, Meals, Notes, Appointments
  • Local E2E encryption; PHI stays on devices
  • Account abstraction for roles & session keys (on Solana)
  • Releases mirrored on Storj; optional IPFS

04 — Privacy & Security

  • No server-side PHI; client-side encryption by default
  • Non-PHI SMS only; view details in the app
  • Device revocation; optional second-device/USB/home-box backups

05 — Market & GTM

  • US: 7M stroke survivors; 53M caregivers
  • Channels: stroke orgs, Facebook groups, AARP, caregiver communities
  • Pricing: $60/year; one-week free trial (first 100 users)

06 — Roadmap

See Release timeline for dates.

07 — Open Source

Open modules (release mirroring, AA session-keys starter) with clear docs; core care engine remains proprietary.

08 — Ask

  • Pilot partners (10–20 families)
  • Grant support & feedback on accessibility
